<b>smart phone တစ်လုံးကို System update တင်လိုက်သောအခါ</b>ဖုန်းတစ်လုံးရဲ့လုပ်ဆောင်နိုင်စွမ်းအားတွေတိုတက်လာပါတယ်။တချို့သော device တွေမှာတော့ ထိုကဲ့သို့ system update တင်လိုက်မှ အခြေနေတွေပိုဆိုးသွားတာမျိုးရှိတက်ပြီး သို့သော်ဤဖြစ်စဉ်သည် လူတစ်ရာမှာတစ်ယောက်သာကြုံတွေ့ရတက်ပါတယ်။phone တစ်လုံးကို <b>System Update</b>တင်တဲ့အခါ အဖြစ်များတဲ့ဆိုးကျိုးတွေတော့ ဖုန်းရဲ့အလုပ်လုပ်ပုံ ပုံမှန်မဟုတော့ဘူး ပိုဆိုးတာက တချို့ app တွေပါသုံးမရတော့ဘူး phone အတွင်းက file တွေပျက်ကုန်တယ် စသည်ဖြင့် အမျိုးမျိုးရှိကြပါတယ်။

ဒါပေမဲ့ ကိုယ်က ဖုန်းအရမ်း ကလိတဲ့ထဲကဖြစ်ပြီးတော့ ကလိတာများသွားပြီ ဖုန်းက ကမောက်ကမဖြစ်နေတယ်။ဖုန်က ဘာမှမလုပ်ပဲပါ၀ါပိတ်ပိတ်သွားတက်တာမျိုးတွေရှိရင်တော့ system update တင်ကြည့်သင့်ပါတယ်။တချို့ဖုန်းတွေ ဆို ကမောက်ကမတွေဖြစ်နေလို့ ဆိုင်သွားပြတယ် firmware ပြန်ရေးရမယ်လို့ပြောပါတယ်။ဆိုင်မပို့ချင်ဘူး OK ဒီလိုအခြေနေတွေမှာ 
System Update တင်သင့်ပါတယ်။ဘာလို့လဲဆိုတော့ System Update ဆိုတာဟာ System တစ်ခုလုံးကို ပြန်လည်ပြင်ဆင်မိုးမံသည့်သဘောဖြစ်ပါတယ်။ထိုကဲ့သို့မိုးမံသည့်အခါ android device တစ်ခုကို မောင်နှင်ရာတွင်အရေးကြီးသော firmware ကို လည်းပြန်လည့်ပြုပြင်မိုးမံပေးခြင်းဖြစ်ပါတယ်။သို့သော် ထိုကဲ့သို့ update တင်ရာတွင် mobile data များစွာအသုုံပြုရန်လိုအပ်သည်ကိုသတိပြုရမှာဖြစ်တာကြောင့် wifi ဖြင့် system update ပြုလုပ်မှသာ အဆင်ပြေမှာဖြစ်ပါသည်။
