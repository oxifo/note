<b>ကွန်ပျူတာ virus မှတ်စု</b>
   ယနေ့ချိန်မှာ လူသုံးအများဆုံးပစ္စည်းတစ်ခုဖြစ်တဲ့ကွန်ပြုတာဟာ နည်းပညာအားဖြင့်မြင်မားတိုးတက်လာသကဲ့သို့ ကွန်ပျူတာကိုအသုံးချတိုက်ခိုက်မှုများလဲတစ်နေ့တခြား ပို၍ပို၍ပေါများလာပါတယ်။ထိုသို့တိုက်ခိုက်ရာတွင် အသုံးချလတ်နက်မှာ virus ခေါ် ကွန်ပျူတာ program တစ်ခုဖြစ်ပြီ ယင် program ရဲ့တိုက်ခိုက်မှုခံရသော ကွန်ပျူတာသည် အချက်လတ်များခိုးယူခံရခြင်း၊virus program ၍စေခိုင်းမှုဖြင့် ပစ်မှုများကျူးလွန်ခြင်း အပြင် ကွန်ပျူတာ စနစ်များအားတိုက်ခိုက်ဖျတ်စီးခံရသောကြောင့် window တက်မလာနိုင်တော့အောင်ပင်ပြင်းထန်နိုင်သည်။
<b>ကူးစက်ခံရခြင်း</b>
    computer virus ကူးဆက်ခြင်းသည်ယေဘုယအားဖြင် အင်တာနက်မှကူးစက်ခြင်းနှင့်virus ထည့်သွင်းထားသော CD ခွေများထည့်သွင်းသုံးပြုခြင်း(fileများမှကူးစက်ခြင်း)တို့မှကူးဆက်နိုင်သည်။
  <b>အင်တာနက်ပေါ်မှ တဆင့်ကူးစက်ခံရခြင်း</b>ဟုဆိုရာတွင် မသာမာသူများတည်ဆောက်ထားသော website များ၀င်ရောက်မိခြင်းမှလည်ကောင်း၊အမည်မသိ Mail များနှင်တွဲကပ်ပါလာသော file များအားဖွင်မိခြင်းမှလည်ကောင်း ကူဆက်လေ့ရှိသည်။
<b>virus ထည့်သွင်းထားသော CDခွေ၊MEMORY STICK များမှကူးစက်ခြင်း</b>သည်လည် အတွေ့များရသော VIRUS တိုက်ခိုက်နည်းတစ်ခုဖြစ်သည်။
